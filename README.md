### Requirement

Check if Python 3 and pip is already installed

### Set up virtual environment

Install pipenv

```sh
$ pip3 install pipenv
```


### Run the server!

Activate the virtual environment

```sh
$ pipenv shell
```
Then

```sh
$ pipenv install
```

Run server

```sh
$ python manage.py runserver
```
